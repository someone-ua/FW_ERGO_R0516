  • Modem Router Mode Wireless and Ethernet connects to the local network, The 4G LTE USB router connects to the Internet.
  • Ethernet Router Mode Wireless and Ethernet connects to the local network, The Ethernet connects to the Internet.
  • Modem Router Mode (preferred) When there is Ethernet Router Mode and Modem Router Mode, preference is given to Modem Router Mode.
  • Ethernet Router Mode (preferred) When there is Ethernet Router Mode and Modem Router Mode, preference is given to Ethernet Router Mode.
  • LTE Bridge Mode Wireless works for access point, AP Clients connects to remote AP, Ethernet connects to the local network.
  • WAN Bridge Mode Wireless works for access point, AP Clients connects to remote AP, Ethernet connects to Internet.
