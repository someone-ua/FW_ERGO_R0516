• Select a MAC filtering mode from the Filtering mode drop-down list box.
  • (1) Disable : Disable MAC filtering.
  • (2) Allow : Allow a client to connect to your device if the client's MAC address is in the MAC Address list.MAC filter adds no more than 16!
  • (3) Deny : Deny a client's access to your device if the client's MAC address is in the MAC Address list.MAC filter adds no more than 16!
