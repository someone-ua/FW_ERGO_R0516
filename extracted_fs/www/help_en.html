Quick Setup

The LTE router Quick Setup tab is designed to assist you in connecting your new LTE router to the Internet. This Quick Setup procedure guides you through step-by-step instructions on how to get an Internet connection up and running.

Screen 1 - User Settings

Via this screen you can change the router's Username and Password.
The default values for username and password are 'admin' and 'admin'

Screen 2 - Internet Connection

The Internet Connection screen allows the user to configure the Internet connection type.

The main Internet connection types are as follows:

  • Disable - No Internet connection. Only the wireless Home Network is active.
  • Cellular - This is the default mode of connectivity. You can insert your SIM into the router and the wireless clients associated can make use of it for internet connectivity! Also note that in this mode, the ethernet port can be used for connecting any LAN side clients. For 3G USBModem, there are a bunch of ISP's that we support - you can select one of them from a dropdown list. Also you can update the router's ISP database by uploading your local ISP settings file.
Screen 3 - Wireless Settings(2.4G)

The Wireless Connection Setup screen contains the Wireless Network Information (WLAN) section which describes the wireless connection status. The user can configure wireless settings and security parameters.

  • 802.11 mode -- Current provide mode 802.11b/g and 802.11b/g/n
    802.11b/g--Allows all the 802.11b or 802.11g devices to connect the router.
  • Channel--Allow router to choose the best channel in the wireless work area,the mode 802.11 depend on if the router works in the 2.4GHZ. the options of the channel show all the effective channels under the mode 802.11
  • Bandwidth -- To control the performance and the interference.Users can configuration it as required.
  • Max connections -- Allows users to config the max connections at the same time.
  • WIFI auto close time -- Allows users to config the time of auto close wifi.
  • Wireless Network Name (SSID) -- User defined name for the wireless network.
  • Wireless Network Password -- User defined password for the wireless network.
  • WPA / WPA2 (most secure) -- Provides maximum level of security. Devices that support WPA or WPA2 will be able to join this network. You must enter a password between 8 and 63 characters.
  • No Security (not recommended) -- Any wireless device can connect your network without entering a password.

PIN Managment

PIN Managment Screen

Enable or disable PIN, and change PIN.

The PIN Managment Screen contains the following fields:

  • PIN/PUK Attempts - display the number of PIN attempts and PUK attempts.The default number of PIN attempts is 3, and PUK attempts is 10; if PIN attempts larger than 3, it will require PUK to unlock the SIM card, and if PUK attempts larger than 10, the SIM card will be abandoned When you input PIN code to enable or disable PIN, if input wrong PIN code, the number of PIN attempts will decrease by 1, same as PUK
  • Enable/Disable PIN - Enable PIN:Input correct PIN code and press the right button, PIN will be enabled successfully, and the right button will display as "Disable PIN". Disable PIN:Input correct PIN code and press the right button "Disable PIN", PIN will be disabled successfully, and the right button will display as "Enable PIN".
  • Change PIN - Enter current PIN, then Enter New PIN and press "Save" button, the PIN will be changed.

DHCP Settings

The Home Network screen displays the DHCP settings and allows users to define IP Addresses range to be assigned to the network devices. The DHCP assigns dynamic IP addresses to devices on a network. DHCP ensures that network devices can have a different IP address every time the device connects to the network. The Home Network screen is used to configure the built-in DHCP Server to assign IP addresses to the computers and other devices on the user's LAN.

DHCP Settings

The DHCP Settings section contains the following fields:

  • DHCP server switch – Users can enable or disable the DHCP server,it will lost the functio of allocation IP address if disable it.
  • DHCP Range -- Selects the basic IP address range, in which the start and the end DHCP IP addresses range are assigned.
  • Start DHCP Address and End DHCP Address -- Defines a range of addresses that the DHCP Server uses when assigning addresses to computers and devices on your Local Area Network. Any addresses that are outside this range are not managed by the DHCP Server; these could, therefore, be used for manually configured devices or devices that cannot use DHCP to obtain network address details automatically.
  • Static IP address -- it can allocate a static IP address for a device.

Wireless Settings(2.4G)

Wireless Network Settings

The Wireless screen allows you to configure the parameters needed for your wireless home network. A host of Security methods are also applicable for this wireless network.

The Settings page contains following fields -

  • 802.11 Mode -- Currently we provide only 802.11b/g mode
  • 802.11b/g -- Allows all 802.11b or g devices (in the 2.4 GHz) to connect to the home router.
  • Channel -- Selects the 802.11 frequency that the home router uses. Allows the home router to select the optimal frequency to use given the radio environment in the vicinity of the home router. The 802.11 standard associates different channel ranges depending on if the home router is operating in the 2.4GHz range. The Channel drop-down list reflects the available channel numbers for the selected 801.11 Mode.
  • Channel Bandwidth -- Enable users to control the trade off between performance and interference with other WiFi users. Users can configure this option as they need.
  • Maximum Simultaneous Clients -- Enable users to control the maximum simultaneous clients connect to router.
  • WIFI Auto OFF Time Settings -- Enable users to control the Interval time to make WIFI off automatically when no client connect to router.

Wireless Security Settings

The Wireless Security section allows users to define the network as well as the security parameters for the wireless home network as required. The factory default setting for security is WPA-WPA2 Mixed mode that allows only the WPA/WPA2 devices to associate with the home router.

The Wireless Security section contains the following fields:

  • Network Name -- Used to change the name of your wireless network. This name appears in window's, Macintosh, Linux or in other known networks.
  • Wireless security
  • NONE -- No wireless security. All traffic may be read by others
  • WPA2-PSK -- The highest Level of Security. Passwords should be at least 8 characters. Only devices that support WPA2-PSK may associate. WPA2 Cipher is AES.
  • WPA-PSK -- The highest Level of Security. Passwords should be at least 8 characters. Only devices that support WPA-PSK may associate. WPA2 Cipher is AES.
  • WPA-WPA2 Mixed -- WPA2-PSK or WPA-PSK Security method depending on the device that needed to connect. Passwords should be at least 8 characters. WPA2 Cipher is AES. WPA Cipher is TKIP/ AES.
  • Password -- A User defined password. Passwords should be at least 8 characters.

WPS Settings

WPS is able to automatically configure Network Name (SSID) WPS and WPA Security Keys for access points and WPS client devices in the network. When connecting to a WPS device, it is not necessary for the user to understand concepts such as SSID and security key.

WPS has two ways to connect:

  • Enter PIN code -- mandatory for WPS certified devices.Enter the PIN code generated by the wireless client and click the Connect button. The wireless gateway will receive an access request from the wireless client within two minutes.
  • Push Button Configuration(PBC) -- This can be a hardware button on the device or a software emulated button (optional for wireless clients).press the WPS buttom on your wireless device, and ther click the Connect button on this page within 2 minutes. The wireles gateway will then receive the access request from the wireless device.

Firewall Settings

IP address filter -- To limit internet access of the computers in LAN by configuring time quantum,data package type,origin IP address,origin port, designated IP address and designated port

Port mapping -- Map the port of the IP address of outside hosts to the inside hosts to provide the relative service.When users visit this port,server will map the request to the correspond device of the LAN automation.

port trigger -- Router will send the outer join to the specified inside port when an application build the outer connection by some specific port

DMZ settings -- It is a buffer between the security system and security system to solve the problems that users of external network can not visit in-house network after install the firewall

domain name filter -- To point some specific strings in the router,once the page you visit contains the strings you pointed,the data package will be abandoned.

Dynamic DNS -- Dynamic DNS (DDNS) is a method of updating domain names in real time.This service is used to provide a domain name for a device that may change its IP address on the network.

Service Access Control -- Service Access Control can limit network traffic and improve network performance, which is the basic means to provide network security Access.
